# AutoLecture
시험자동화 TC 문서 및 Postman 스크립트 자동화


## Start
**간단한 프로그램으로 예외처리가 없으므로 입력시 주의**
1. ID 입력
2. 패스워드 입력
3. 강의 선택
4. 주차 선택
5. 몇 강부터 시작할 것 인지 선택
